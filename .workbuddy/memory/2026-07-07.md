### 推荐架构变更
原方案：HTML 爬虫 → 转 MD → LLM 全量处理
新方案：CBP JSON API（免认证）→ Pass 1 JSON 字段直读 → Pass 2 LLM 边缘补漏 → 校验 → SQLite
优点：复杂度降低 70%+、成本可控、无需反爬对抗

### 关键决策
- 使用 CBP JSON API 替代 HTML 爬取
- 两阶段管道：正则直读（80% 通过） + LLM 边缘补漏（~20%）
- LLM 选型：Qwen-Turbo（性价比最优，国内直连）
- 数据库：SQLite（221k 条完全够用）
